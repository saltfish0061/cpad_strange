# Project: Online Food & Drink Ordering System

## Overview
This is a web-based Online Food & Drink Ordering System designed to centralize and streamline the ordering process for cafeteria vendors, students, and staff. The system addresses issues such as long waiting times, lack of centralized management, and order inaccuracies.

## Features

## Tech Stack
- **Frontend**: HTML5, CSS3
- **Backend**: PHP
- **Database**: MariaDB 

## Setup Instructions
1. Clone the repository into your XAMPP `htdocs` folder.
2. Start Apache and MySQL from the XAMPP Control Panel.
3. Go to phpMyAdmin, create an empty database named exactly cpad_03_Strange.
4. Go to the "Privileges" tab and create a user named cpad with the password cpadPassword, and check the box to grant it all privileges.
5. Click on the cpad_03_Strange database on the left sidebar, click the Import tab at the top, choose the cpad_03_Strange.sql file from the project folder, and hit "Go".
6. Open `http://localhost/your-repo-name` in your browser.
